# RTL2GDSII Workshop

## Overview
This project demonstrates the complete RTL to GDSII flow for a digital design. It covers all major stages of the VLSI design process, starting from RTL coding to final physical design outputs.

##  Flow
- RTL Design (Verilog)
- Functional Simulation
- Synthesis
- Floorplanning
- Placement & Routing
- Signoff (DRC/LVS, Timing Analysis)

##  Project Structure
- `rtl/` → Verilog design files  
- `testbench/` → Simulation files  
- `scripts/` → TCL scripts for automation  
- `synthesis/` → Synthesized netlist  
- `place_route/` → PnR outputs  
- `reports/` → Timing, area reports  
- `results/` → Final outputs (GDSII, netlist)

## Tools Used
- Design Compiler  
- ICC2  
- PrimeTime  

## Objective
To understand and implement a complete ASIC design flow using industry-standard tools and structured methodology.
